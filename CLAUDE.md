# CLAUDE.md — Sapinover-Overnight-Lunar

## What This Is
Standalone Lunar New Year 2026 event study dashboard. Analyzes overnight ATS behavior during the Asian market closure period. Static HTML on GitHub Pages. No authentication.

## Live URL
https://ejcacciatore.github.io/Sapinover-Overnight-Lunar

## Scope
- 16 sessions analyzed (On / Off / On pattern)
- 251,600 observations across 3 venues (BlueOcean, Bruce, Moon)
- Four phases: Baseline, Pre-Holiday, LNY Holiday, Recovery
- +47% volume recovery documented post-holiday
- Self-contained: all data embedded in HTML/JS, no external API calls

## Stack
Static HTML, Chart.js, vanilla JS. No build step. No encryption. Public access.

## Deployment
Push to main branch. GitHub Pages auto-deploys.
